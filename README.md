# NLP & Knowledge Graphs Projects

This repository contains a collection of individual projects developed throughout a learning journey in Natural Language Processing (NLP) and Knowledge Graphs (KG). Each project contributes to a growing toolbox for building real-world NLP applications.

## Projects Overview

### 1. Sentiment Analysis API  
A web service built with FastAPI and Docker that performs sentiment analysis on text input. Focused on API design, deployment, and NLP pipelines.

### 2. PDF to Sentences  
A system that extracts and converts content from PDF files into structured sentences, enabling further text processing and analysis.

### 3. Named Entity Recognition (Text-to-Person)  
An application that uses an LLM API to identify and extract person names from text, with emphasis on prompt engineering and model interaction.

### 4. Entity Linking (Person-to-Knowledge Graph)  
Maps extracted entities to entries in a knowledge graph (e.g., Wikidata) and retrieves structured information using SPARQL queries.

### 5. Information Retrieval System  
Implements search techniques (sparse, dense, and hybrid) to retrieve relevant documents from a corpus based on user queries.

### 6. Retrieval-Augmented Generation (RAG)  
Combines retrieval methods with language models to generate answers grounded in a specific dataset or document collection.

### 7. Text-to-Query  
Transforms natural language questions into structured queries for knowledge graphs, enabling automated data access.
